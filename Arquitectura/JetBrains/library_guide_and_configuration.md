
## Librerías Especializadas

### Visualización de Datos (Recharts)
- Gráficos interactivos
- Line charts, bar charts, pie charts
- Responsive
- Personalizable

### Exportación de Datos

#### jsPDF + jsPDF-autotable
- Generación de PDFs del lado del cliente
- Tablas automáticas
- Personalización de estilos
- Logos e imágenes

#### XLSX
- Exportación a Excel
- Múltiples hojas
- Formato de celdas
- Fórmulas

### Notificaciones (React Hot Toast)
- Toasts no intrusivos
- Personalizable
- Promise-based
- Animaciones suaves

### Iconos (Lucide React)
- Librería moderna de iconos
- Tree-shakeable
- Consistente
- Amplia colección

## Configuración y Build

### Vite
Build tool ultrarrápido:
- Hot Module Replacement (HMR) instantáneo
- Build optimizado con Rollup
- Plugin de React oficial
- TypeScript nativo
- Entorno de desarrollo rápido
