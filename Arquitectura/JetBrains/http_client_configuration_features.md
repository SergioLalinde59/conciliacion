### 4. **Gestión de Estado**

#### React Query (TanStack Query)

- **Mutations**: Operaciones de escritura
- **Invalidation**: Actualización automática de caché
- **Optimistic Updates**: Actualizaciones optimistas
- **Background Refetching**: Actualización en segundo plano
