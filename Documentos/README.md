# Conciliación

Sistema integral de conciliación bancaria y gestión de movimientos financieros.

## Características
- Procesamiento de archivos Bancolombia (XML/CVS).
- Gestión de movimientos de tarjetas de crédito y cuentas de ahorros.
- Reportes interactivos con drill-down de tres niveles.
- Visualización de tendencias y comparativas.

## Estructura del Proyecto
- `frontend/`: Aplicación React + Vite.
- `Backend/`: API y servicios de procesamiento de datos.
- `data/`: Almacenamiento de archivos de entrada y procesados.
