### Contexto del Proyecto
Este archivo pertenece al proyecto **ConciliacionWeb**, un sistema de conciliación bancaria que incluye:

#### Características Principales
- **Arquitectura:** Hexagonal (puertos y adaptadores)
- **Backend:** FastAPI con Python
- **Frontend:** React con TypeScript
- **Base de Datos:** PostgreSQL
- **Infraestructura:** Docker

#### Funcionalidades Implementadas (Según Historial)
1. **Extracción de Movimientos PDF**

   - Validación de moneda (COP/USD)

2. **Gestión de Conciliaciones**
   - Vista mensual de conciliaciones
   - Comparación entre extractos y sistema
   - Manejo de movimientos en pesos y dólares

3. **Sistema de Matching Inteligente** (En desarrollo)
   - Configuración de matching
   - Vinculación de movimientos
